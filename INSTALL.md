# Android 安装说明

`钱去哪了` 当前面向自己和少量熟人使用，通过 GitHub Release 下载 APK 安装。数据只保存在本机，不会自动同步到云端。

## 安装 APK

1. 打开项目的 GitHub Releases 页面。
2. 下载最新版本里的 `ledger-journal-android-x.y.z.apk`。
3. 在 Android 手机上打开 APK 文件。
4. 如果系统提示禁止安装，请进入提示页开启“允许安装未知应用”。
5. 安装完成后打开 `钱去哪了`，注册一个本机账号。

## 第一次使用

1. 先记一笔测试账目，确认支出、收入、分类和账户都能正常保存。
2. 到“设置”里点“导出备份”，把 JSON 备份保存到微信文件、网盘或电脑。
3. 后续换手机、卸载 App、清除应用数据前，都先导出备份。

## 恢复备份

1. 打开“设置”。
2. 点“导入备份”。
3. 粘贴完整 JSON 备份内容。
4. 确认恢复后，账目、账户、模板、分类和预算会从备份恢复。

## 更新版本

1. App 设置页点“检查 GitHub Release”。
2. 有新版本时下载新的 APK。
3. 直接安装新 APK 覆盖旧版本，正常情况下本机数据会保留。
4. 更新前仍建议先导出一次备份。

## 注意事项

- 不要把 APK 发给不熟悉来源的人安装。
- 不要删除或清空应用数据，除非已经确认备份可用。
- 当前版本不提供云同步、找回密码、生物识别或跨设备自动恢复。
